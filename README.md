#### Created with: 
A game created in C# Windows Forms using Visual Studio 2010 Express Edition in my first year of college (2013/2014). 

#### Description: 
A simpler version of the classic Space Invaders.

#### Controls: 
A and D to move left and right.
Space to fire.
